How To Fill The Form ?

First Name

•  You Have To Enter Your First Name In Alphabets Only In Either Caps or Small

Eg:John
Numeric and special symbols will tend to error displaying : "Enter A Proper First Name"

Last Name

•  You Have To Enter Your Last Name In Alphabets Only In Either Caps or Small

Eg:Hopkins
Numeric and special symbols will tend to error displaying : "Enter a proper Last Name"

Email

• The Email Address Should contain '@'. If ‘@’ is not in the given input. An error will be displayed as :“Please include ‘@’ in the Email Address”
After '@' , You Should provide a proper email address. Otherwise an error will be displayed : "Enter A Valid Email Address"

Eg:johnhopkins@gmail.com

Date of Birth / Age

• Your Age Should Be Between 18 and 34 You Have to select the date of birth in the given calender field. If Your Age Is Below 17 Or Greater Than 34 , You Are InEleigible To The Scheme.

Gender

• You Have To Select Either One Of The Given Options

Certificate No

• You Should Enter Valid Certificate Number Which Will Be In The Top Right Corner Of Your Caste Certificate. Invalid Caste Cert. No input will result in error which displays : "Enter Valid Certificate Number."

Education

• You Have To Select Your Current Educational Qualification From The Drop down menu

Note:

Your Current / Pursuing Educational degree is the option you must select

Eg:If You Are Studying / Studied any Bachelors program select : Undergraduate
If You are studying / Studued any masters program , select : Graduate
If You are Studying / Studued any Doctorate Program , select : Post-Graduate

States

• A Drop Down Will Displays All 29 States & 7 Union Territories Of India. You Must Select The State / U.T To Which You Belong To.

Cert. File Upload

• You Have to Upload The pdf file of your Caste certificate to submit your details to register in the scheme.
